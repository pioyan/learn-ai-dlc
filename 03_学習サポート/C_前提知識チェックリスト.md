# C: 前提知識チェックリスト

AI-DLC を学ぶ・実践するために、どの知識がどの程度必要かを確認するチェックリスト。  
「知らなくてもOK」なものと「必須」なものを明確に分けています。

---

## 使い方

1. 各項目を読んで、現在の理解レベルに ✓ を入れる
2. 「必須」なのに「知らない」がある → その補足資料を先に学ぶ
3. 「あると有利」は AI-DLC を学びながら並行して学んでも良い

---

## 分類凡例

| 記号 | 意味 |
|------|------|
| 🔴 必須 | これがないと AI-DLC の実践事例を理解できない |
| 🟡 あると有利 | あれば理解が深まる。なくても基本は学べる |
| 🟢 上級者向け | Phase 3（実践）になってから学んでもOK |

## 結果の解釈

### 🔴 必須項目が「知らない」が 3 つ以上ある場合
→ AI-DLC の学習と並行して、補足資料を優先的に学ぶことを推奨  
→ 特に「アジャイル基礎」と「AI ツール体験」は先に済ませると効果的

### 🔴 必須項目はすべて「知っている」場合
→ フェーズ1 からすぐ開始できます

### 🟡 有利項目に「知らない」が多い場合
→ 実践例を読みながら分からない言葉が出たら [B_用語集.md] を参照  
→ DDD については実践例_02 を読む際に用語集と並行して確認

## 学習優先順位（知らない場合の補強順）

```
1. アジャイル基礎 → Scrum ガイドを 1 時間で通読
2. AI ツール体験 → ChatGPT/Claude に簡単な質問を 10 個投げてみる
3. Python 基礎 → Paiza や Progate で 2〜3 時間
4. DDD 基礎 → 用語集の DDD セクションを読んでから実践例_02 を読む
5. AWS 基礎 → AWS の「はじめての AWS」ページで 1 時間
```
